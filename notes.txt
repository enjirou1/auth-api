Buat Cluster RDS
- Engine options, pilih engine PostgreSQL, pilih versi dibawah 15
- Choose a database creation method Full configuration
- Isi DB instance identifier, Master username, Master password
- Uncheck Enable storage autoscaling
- Public access No
- Pilih security group database
- Uncheck Turn on Performance Insight
- Uncheck Enable automated backups
- Enable auto minor version upgrade

- ssh -i "<key>.pem" <user>@<alamat instance EC2>
- Kalau error UNPROTECTED PRIVATE KEY FILE
  icacls "forum-api-webserver.pem" /remove "Authenticated Users"
  icacls "forum-api-webserver.pem" /remove "Users"
  icacls "forum-api-webserver.pem" /remove "Everyone"
  icacls "forum-api-webserver.pem" /inheritance:r
  icacls "forum-api-webserver.pem" /grant:r %username%:R

Setup Project
- git clone https://github.com/enjirou1/auth-api.git
- cd auth-api
- npm install
- touch .env
- vim .env, tekan tombol insert untuk edit lalu esc untuk keluar mode kemudian :wq untuk keluar
- mkdir -p config/database
- touch config/database/test.json
- vim config/database/test.json
- npm run migrate up
- npm run migrate:test up
- npm run test
- pm2 start npm --name "auth-api" --run "start"
